# AIBlock
A repository for a custom browser extension to hide AI overviews from search results

**AIBlock does not prevent the use of AI, it simply hides these elements from your search results**

## Installation
AIBlock is not currently available on any browser's web store. To use, follow the installation guide below:

Download the repository:
```
git clone https://github.com/DavyJonesStockings/AIBlock.git
```

Go to `chrome://extensions`. Enable Developer Mode and select `Load Unpacked`. Select the folder you used to clone this repository.
